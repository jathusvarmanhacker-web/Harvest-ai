
A smart AI-powered agriculture assistant built with Streamlit. Helps farmers reduce crop loss, improve harvest quality, and make better farming decisions.

## Features

- **Dashboard** — Live weather, crop status, and AI insights
- **Crop Scanner** — Upload leaf photos to detect diseases and get treatment advice
- **AI Assistant** — 24/7 farming chatbot with crop-specific answers
- **Market Prices** — Real-time crop price tracking with AI insights
- **Harvest Planner** — AI-powered harvest date and yield prediction

## Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/agroshield-ai.git
cd agroshield-ai

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run app.py
```

Then open http://localhost:8501 in your browser.

## Deploy to Streamlit Cloud (free)

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click **New app** → select your repo → set `app.py` as main file
4. Click **Deploy** — done!
